# User Intent

## Target Audience
Machine learning practitioners, AI researchers, and local LLM enthusiasts who want to remove safety alignment from transformer-based language models without expensive post-training or deep transformer expertise.

## Use Cases
- Decensoring open-source LLMs for unrestricted local use
- Researching model internals and interpretability through residual vector analysis
- Comparing abliteration quality across models and parameter configurations

## Documentation Focus
This documentation takes the form of a **practical user guide with technical depth**:
- Quick start and installation for immediate use
- Comprehensive configuration reference for all tunable parameters
- Explanation of the technical approach (directional ablation, TPE optimization)
- Research features guide (residual plots, geometry analysis)
- Supported model architectures and hardware considerations
